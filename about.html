About

Trendulo enables trending and real-time patterns of life analysis using Twitter messages. Trendulo began as a demo for the Introduction to Apache Accumulo presentation at the Boulder/Denver BigData Meetup. Trendulo is heavily inspired by other Twitter analytic tools such as timeu.se and Trendistic. Twitter's Streaming API has been used to collect tweets from the United States since Christmas of 2011.

Trendulo is built using many free and open source technologies including:

  • Apache Accumulo
  • Apache Hadoop
  • Twitter Streaming API
  • Twitter4J
  • Spring Framework
  • jQuery
  • Bootstrap
  • Datepicker for Bootstrap
  • ICanHaz.js
  • Highcharts JS
  • History.js
  • nginx
  • Apache Tomcat

Trendulo is developed by Jared Winick who can be found on Twitter or LinkedIn

Trendulo can be found on github at https://github.com/jaredwinick/Trendulo. Please feel free to submit any issues you find.
